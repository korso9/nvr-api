Backend API Docs

User Authentication

All routes for user authentication

POST

- Login

Log the user in and return JSON Web Token

https://www.neurologicavr.com/api/users/login

BODY

- JSON

{ "emailAddress": "", "password": "" }

POST

- Register

Register a new user

https://www.neurologicavr.com/api/users/register/

BODY

- JSON

{ "emailAddress": "", "firstName": "", "lastName": "", "password": "" }

PUT

- Confirm Email

Confirm user email

https://www.neurologicavr.com/api/users/confirm/:id

PATH PARAMS

id

- User's account id

BODY

- JSON

{ "verificationCode": "" }

PUT

- Forgot Password

Request to initiate password reset

https://www.neurologicavr.com/api/users/forgot

BODY

- JSON

{ "emailAddress": "" }

PUT

- Reset Password

Reset user password

https://www.neurologicavr.com/api/users/reset/:id

PATH PARAMS

id

- User's account id

BODY

- JSON

{ "verificationCode": "", "newPassword": "" }

GET

- Resend Verification Code

Reset and send new verification code

https://www.neurologicavr.com/api/users/resend/:id

PATH PARAMS

id

- User's account id

Scores

All routes for user quiz scores

POST

- Post New Score

Add a new score for a specific product and lesson

https://www.neurologicavr.com/api/scores

AUTHORIZATION

- Bearer Token

BODY

- JSON

{ "product": "", "lesson": "", "newScore": "" }

GET

- Get Score

Get the score of a specific lesson

https://www.neurologicavr.com/api/scores/:product/:lesson

AUTHORIZATION

- Bearer Token

PATH PARAMS

product

- Product as string (e.g. 'vds')


lesson

- Lesson number
